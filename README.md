# Smart Surveillance with AI

This project is an AI-powered surveillance system that uses YOLOv8 for object detection, activity monitoring, and real-time alerts.

## Features
- Detects people, vehicles, and objects in real-time
- Monitors restricted zones for intrusion
- Sends alerts when suspicious activity is detected
- Web dashboard for monitoring alerts
- Streamlit dashboard for visualization

## Installation
```bash
pip install -r requirements.txt
```

## Usage
Run the monitoring script:
```bash
python monitor.py
```

Run the Flask app:
```bash
python app.py
```

Run the Streamlit dashboard:
```bash
streamlit run streamlit_dashboard.py
```
